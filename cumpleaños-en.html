Birthday Reminder

Project Details

Description: An interactive web application designed to celebrate birthdays in a unique way. Users can access a personalized welcome page with an animated background, where they can see a special message, interact with a celebration button, and sign up for more features.

Technologies Used: Tailwindcss, Laravel, React, MySQL

Features:

  • Animated birthday background with gradients and bright colors.
  • Real-time interaction with a button to celebrate the event.
  • Redirection to customized views based on user authentication status.
  • Responsive design using Tailwind CSS, ensuring a smooth user experience on both mobile and desktop devices.
