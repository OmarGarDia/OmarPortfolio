Shopping Cart

Project Details

Description: A small React project that loads product data from an API and allows users to add items to a shopping cart. Users can increase, decrease, or remove items, and the total price is updated in real time.

Technologies Used: React, Bootstrap, Material UI

Features:

  • Loads products from an API.
  • Add products to the shopping cart.
  • Increase or decrease product quantity.
  • Remove products from the cart.
  • Real-time total price calculation.
  • Responsive interface with Bootstrap and Material UI.
