Carrito Compra

Detalles del Proyecto

Descripción: Un mini proyecto de React donde se cargan Datos de una API de productos, y se pueden añadir a un carrito. En el carrito se pueden añadir mas, menos, quitar y se ve el total

Tecnologías Utilizadas: React, Bootstrap, MAteria UI

Características:

  • Carga de productos desde una API.
  • Añadir productos al carrito.
  • Incrementar o disminuir la cantidad de productos.
  • Eliminar productos del carrito.
  • Visualización del total de la compra en tiempo real.
  • Interfaz responsiva con Bootstrap y Material UI
